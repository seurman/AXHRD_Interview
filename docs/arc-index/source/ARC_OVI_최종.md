**OVI --- Organization Velocity Index**

우리 조직은 점점 좋아지고 있는가?

*ARC Index 구성 도구 3/3 · 최종 검토 확정판*

+------------------+--------------------------------------------------+
| **문항 구성**    | HV 조직건강속도 5문항 + CV 변화실행속도 5문항 +  |
|                  | AV AX전환속도 5문항 = 15문항                     |
+==================+==================================================+
| **핵심 원리**    | 회고적 질문: \'지금 어떤가\'가 아닌 \'6개월      |
|                  | 전보다 어떻게 달라졌는가\'를 묻는다              |
+------------------+--------------------------------------------------+
| **응답 방식**    | 5점 단일응답 (변화 방향·속도 전용 척도) /        |
|                  | 소요시간 약 5분                                  |
+------------------+--------------------------------------------------+
| **척도 의미**    | ① 많이 악화 ② 다소 악화 ③ 변화없음 ④ 다소 개선 ⑤ |
|                  | 크게 개선                                        |
+------------------+--------------------------------------------------+
| **분석 결과**    | OVI 영역별 점수 · Dynamic Congruence · Risk      |
|                  | Index(HV) · AX속도 갭                            |
+------------------+--------------------------------------------------+
| **OVI의 차별점** | Nadler & Tushman이 정의하지 않은 동적            |
|                  | 정렬(Dynamic Congruence) 측정                    |
|                  |                                                  |
|                  | CV01이 가장 핵심 --- 결정→현장 적용 속도를 직접  |
|                  | 측정                                             |
+------------------+--------------------------------------------------+

  --------------- -----------------------------------------------------------------------------------
  **비교 기준**   \'지금 어떤가\'가 아닙니다. \'지난 6개월 전과 비교해서 어떻게 달라졌는가\'입니다.
  **CV01 예외**   CV01은 속도 자체를 묻습니다: ① 매우느림 ② 느린편 ③ 보통 ④ 빠른편 ⑤ 매우빠름
  **AV05**        외부 환경 대비 우리 속도를 묻습니다. 절대적 변화가 아닌 상대적 속도입니다.

  **HV04**   지난 6개월과 비교해서 나의 성장과 발전 기회가 더 많아지고 있다고 느낀다     ① ② ③ ④ ⑤

  **CV02**   지난 6개월과 비교해서 우리 조직이 외부 변화에 더 빠르게 대응하고 있다        ① ② ③ ④ ⑤

**OVI 해석 기준**

  **OVI 점수**     **판 정**          **경영진 메시지 · 처방**
  ---------------- ------------------ ---------------------------------------------------------------------------
  **4.5 \~ 5.0**   **빠른 개선 ↑↑**   올바른 방향으로 가속 중. 모멘텀을 유지하고 병목 영역을 집중 관리하십시오.
  **3.5 \~ 4.4**   **개선 중 ↑**      변화가 진행 중입니다. 속도를 더 내려면 의사결정 병목을 확인하십시오.
  **2.5 \~ 3.4**   **정체 →**         변화가 멈췄습니다. CV01(실행속도)과 AV05(AX속도)를 즉시 확인하십시오.
  **1.5 \~ 2.4**   **악화 중 ↓**      조직이 뒤로 가고 있습니다. Risk Index와 함께 긴급 개입이 필요합니다.
  **1.0 \~ 1.4**   **급속 악화 ↓↓**   즉각적 위기 대응이 필요합니다. OHI와 ORI를 함께 긴급 점검하십시오.
